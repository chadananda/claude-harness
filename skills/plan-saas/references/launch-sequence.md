# Launch Sequence Reference

## Phase 1: Foundation (Weeks 1-4)

### Week 1: Setup
- [ ] Landing page live with value prop + waitlist form
- [ ] Analytics installed (Plausible/PostHog — not Google Analytics, privacy matters)
- [ ] Email capture system active (ConvertKit, Loops, or Resend)
- [ ] Social profiles created and branded (Twitter/X, LinkedIn minimum)
- [ ] Build-in-public first post: "I'm building [product] because [story]. Follow along."

### Week 2: Content Foundation
- [ ] Write pillar page #1 (target MOFU keyword)
- [ ] Write 2 BOFU articles (comparison page + alternative page)
- [ ] Set up editorial calendar for next 10 weeks
- [ ] First community engagement posts (answer questions, no self-promotion yet)

### Week 3: Community & Outreach
- [ ] Active in 3-5 communities daily (30 min/day)
- [ ] Identify 50 potential beta users from community engagement
- [ ] Write pillar page #2
- [ ] Write 2 MOFU articles (how-to guides)
- [ ] First cold outreach batch (10-15 personalized emails)

### Week 4: Pre-Launch Prep
- [ ] Landing page updated with social proof (waitlist count, testimonials from beta conversations)
- [ ] Product Hunt draft created (all assets: logo, screenshots, video, tagline, description)
- [ ] Referral waitlist mechanics active
- [ ] Write 2 more articles
- [ ] Second cold outreach batch
- [ ] Waitlist update email sent

## Phase 2: Pre-Launch (Weeks 5-7)

### Week 5: Beta Access
- [ ] Beta invitations sent to top 20-30 waitlist members
- [ ] Feedback collection system active (Typeform, Canny, or simple email)
- [ ] Write 2 articles + 1 case study from beta feedback
- [ ] Social proof: share beta user quotes (with permission)
- [ ] Community presence deepened — you should be a recognized name by now

### Week 6: Launch Assets
- [ ] Product Hunt listing fully polished (get feedback from 2-3 people)
- [ ] Launch email sequence drafted (announcement + reminder + launch day + follow-up)
- [ ] Social media launch content pre-written (day-of thread, posts for each platform)
- [ ] Influencer outreach: contact 15-20 micro-influencers with free access offer
- [ ] Write 2 more articles

### Week 7: Final Prep
- [ ] Pricing page live
- [ ] Billing integration tested end-to-end (run a real test transaction)
- [ ] Onboarding flow tested with 3+ beta users watching screen share
- [ ] All launch emails scheduled
- [ ] Community launch posts drafted for each community
- [ ] Hunter secured for Product Hunt (or self-post plan confirmed)

## Phase 3: Launch (Week 8)

### Launch Day (hour-by-hour)
- **12:01 AM PT**: Product Hunt goes live (or schedule for midnight)
- **6:00 AM**: Email blast to full waitlist: "We're live!"
- **7:00 AM**: Social media posts go live (Twitter thread, LinkedIn post)
- **8:00 AM**: Post in communities (only ones where you've been active)
- **9:00 AM - 6:00 PM**: Respond to EVERY Product Hunt comment within 1 hour. Engage on social.
- **12:00 PM**: Second social push — share early reactions, upvote count, user quotes
- **6:00 PM**: Thank-you post with day-1 numbers (transparent, build-in-public style)
- **All day**: Monitor for bugs/issues. Fix immediately. Nothing kills launch momentum like downtime.

### Launch Week (Days 2-7)
- [ ] Follow up with every Product Hunt commenter via email/DM
- [ ] Daily social updates with launch metrics
- [ ] Respond to all new user feedback within 4 hours
- [ ] Second email to waitlist with launch results + social proof
- [ ] Influencer follow-up: check if they've tried the product, offer help
- [ ] Write "launch retrospective" blog post (great build-in-public content)

## Phase 4: Post-Launch (Weeks 9-12)

### Week 9: Analyze & Iterate
- [ ] Full launch metrics review: signups, trials, conversions, traffic sources, churn
- [ ] Identify top 2 performing channels → double down on these
- [ ] Identify bottom 2 performing channels → reduce or cut
- [ ] Collect NPS from first paying customers
- [ ] Write 2 articles

### Week 10: Scale What Works
- [ ] Increase investment in top 2 channels (more content, more community, or start paid)
- [ ] Launch referral program (if 50+ active users)
- [ ] First partnership outreach (complementary tools)
- [ ] Write 2 articles + 1 case study

### Week 11: Expand
- [ ] Test paid acquisition on one channel ($500 test budget)
- [ ] Guest post on industry publication
- [ ] Webinar or co-marketing event with partner
- [ ] Write 2 articles

### Week 12: Review & Plan
- [ ] Full 12-week retrospective: what worked, what didn't, key numbers
- [ ] Set 90-day OKRs based on learnings
- [ ] Plan next quarter content calendar
- [ ] Evaluate: expand team? Raise capital? Bootstrap further?
